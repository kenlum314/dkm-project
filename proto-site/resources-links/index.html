Resources & Links

Below you will find resources & links relevant to the course as well as ones that are generally applicable to any BCIT student.

Course Resources & Links

  • Google Search Engine — A search engine that facilitates searching for information by keywords, often leading to one of the websites below.
  • Stack Overflow — A question and answer site for programmers. Most programming questions have already been answered here, so often there is no need to even post a question!
  • W3Schools — A good reference for the most common web technologies
  • JSON Formatter & Validator — An online formatter and validator for JSON, which is the most common data serialization format used to communicate with web service APIs.
  • Java API Documentation — Documentation for what is perhaps the most prolific modern programming language.
  • The Groovy Programming Language — Documentation for the Groovy programming language; an extension of Java.
  • The Grails Framework — A reference for the Grails web application framework.
  • Handlebars.js — The website for the Handlebars JavaScript templating engine.
  • Underscore.js — Underscore.js is a library of functions for the processing and manipulating of JavaScript data structures.
  • Node.js — Node.js is a popular server-side JavaScript runtime environment, one of several which enable JavaScript to finally be used on both the client and server side.
  • Meteor Framework — The website for the Meteor JavaScript web development framework.
